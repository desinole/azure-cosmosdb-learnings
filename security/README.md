# Security & Compliance

This folder contains learnings about security, access control, and compliance in Azure Cosmos DB.

## Topics Covered

- Authentication and authorization patterns
- RBAC and fine-grained access control
- Network isolation and private endpoints
- Encryption at rest and in transit
- Key management and rotation
- Audit logging and compliance
- Managed identities integration
- Customer-managed keys (CMK)
- Security best practices

## Document Template

When adding a new learning, consider including:

1. **Title**: Clear, descriptive title
2. **Security Requirement**: What was the security goal?
3. **Implementation**: How was security implemented?
4. **Configuration**: Security settings and policies
5. **Validation**: How was security verified?
6. **Compliance**: Any compliance requirements met
7. **Trade-offs**: Performance or operational impacts
8. **Recommendations**: Security best practices
