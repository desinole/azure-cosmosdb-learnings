The learnings are organized by the following product areas:

### [Monitoring](./monitoring/)
Explore monitoring tools, metrics, alerts, diagnostics, and observability best practices for Azure Cosmos DB.

### [Performance Optimization](./performance-optimization/)

### [Security & Access Control](./security-access-control/)
