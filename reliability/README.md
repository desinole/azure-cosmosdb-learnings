# Reliability

This folder contains learnings related to ensuring high availability and reliability in Azure Cosmos DB.

## Topics Covered

- Multi-region configurations and failover strategies
- Consistency level selection and trade-offs
- Retry policies and error handling patterns
- Service limits and quotas management
- Backup and restore strategies
- Regional outage handling
- Session token management
- Conflict resolution policies
- Service health monitoring

## Document Template

When adding a new learning, consider including:

1. **Title**: Clear, descriptive title
2. **Scenario**: What reliability challenge was faced?
3. **Configuration**: Region setup, consistency level, etc.
4. **Issue**: What went wrong or what was the concern?
5. **Resolution**: How was reliability improved?
6. **Code Examples**: Relevant implementation details
7. **Validation**: How was the solution tested?
8. **Recommendations**: Best practices and lessons learned
