# Performance

This folder contains learnings and best practices related to Azure Cosmos DB performance optimization.

## Topics Covered

- Request Unit (RU) optimization and capacity planning
- Query performance tuning and optimization techniques
- Indexing strategies and composite indexes
- Partition key selection and design patterns
- Bulk operations and batch processing
- SDK optimization and connection pooling
- Cross-partition query optimization
- Latency reduction techniques
- Hot partition prevention and mitigation

## Document Template

When adding a new learning, consider including:

1. **Title**: Clear, descriptive title
2. **Problem/Scenario**: What performance issue was encountered?
3. **Environment**: SDK version, API type, consistency level, etc.
4. **Analysis**: How was the issue identified?
5. **Solution**: What fixed or improved the performance?
6. **Code Examples**: Before/after code snippets
7. **Results**: Metrics showing improvement
8. **Key Takeaways**: Lessons learned and recommendations
